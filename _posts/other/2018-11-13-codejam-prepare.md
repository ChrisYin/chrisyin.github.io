+ 比较排序
  + bubble sort
    + 复杂度: O(n^2)
    + 最大的元素通过相邻的元素交换,换到数组的最后
    + 最坏情况 n的平方次完成排序;
  + select sort
    + 复杂度: O(n^2)
    + 选出最小元素,交换一次到数组最前面
  + insertion sort
    + 复杂度: O(n^2)
    + 分成两个部分:排序部分和乱序部分;每次比较乱序部分第一个元素与排序部分,插入进入已经排序好的数组;
  + quick sort
    + 复杂度: O(nlogn)
    + 最好情况是二分法;最差情况是选择排序了;
  + heap sort
    + 复杂度: O(nlogn)
